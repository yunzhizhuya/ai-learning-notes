# FDE第2-3周学习笔记：提示词从入门到进阶

> 时间：2026年6月9日-12日
> 阶段：FDE第1个月·第2-3周

## 一、四段式提示词框架

写提示词就四个模块，缺一不可：

```
【角色】你是谁？专长是什么？
【规则】允许做什么、怎么输出、禁止做什么
【示例】输入→输出，至少1个
【约束】格式、长度、边界条件（进阶才加）
```

**核心原则：提示词越具体，AI输出越稳定。模糊=失控。**

## 二、Coze Bot搭建经验（3个实战Bot）

### 1. 闲鱼AI客服Bot
- 用途：自动回复闲鱼客户咨询
- 关键：提示词里写清商品信息、价格、交付流程
- 技巧：禁止闲聊规则必须有，否则Bot会被带偏

### 2. 抖音内容生产Bot
- 用途：输入选题，输出文案+配图建议+发布时间
- 关键：输出格式要约束，否则文案长短不一
- 技巧：加长度限制，文案控制在可拍摄范围内

### 3. AI数据分析Bot
- 用途：输入数据，输出分析建议
- 关键：强制JSON输出，方便后续自动化对接
- 技巧：格式强约束+禁止多余解释

**共同经验：**
- 先在对话框调试提示词，满意了再创建Bot
- Bot上线后用5-10条真实输入测试，根据跑偏情况调整规则
- 禁止规则比允许规则更重要，先写"不能做什么"

## 三、提示词三大进阶技巧

### 技巧1：角色锁定

**目的：** 让AI严格扮演特定角色，不跑偏

**套路：** 身份 + 允许范围 + 拒绝规则

**实战：工程塑料检测专家**
```
【角色】工程塑料检测专家，专精POM/COMP/SPP，10年实验室经验
【规则】
1. 只回答检测相关问题（MI值、标准、操作规范、设备校准、数据判定）
2. 不确定的明确说"需查证"，不编造数据
3. 闲聊/无关话题统一回复"此问题超出检测专业范围，无法回答"
【约束】回答不超过3句话，不用客套话
```

**适用场景：** 客服Bot、专业咨询Bot、行业问答Bot

**关键点：**
- 拒绝话术必须写死一句话，不给AI发挥空间
- "允许范围"要列具体关键词，不能写"相关工作"这种模糊词

---

### 技巧2：输出格式强约束

**目的：** 控制AI输出结构，方便程序解析

**套路：** 固定JSON结构 + 禁止JSON以外的任何内容

**实战：抖音视频数据分析**
```
【规则】
1. 回复必须是且仅是一个JSON对象
2. JSON结构严格如下：
   {"play_rate":"互动率","content_type":"内容类型判断","weak_point":"最薄弱环节","suggestion":"一句话建议≤20字"}
3. 严禁在JSON前后添加任何文字、解释、markdown标记
4. 严禁输出上述4个字段以外的内容
```

**适用场景：** 数据分析、自动化流程对接、表格写入

**关键点：**
- "必须是且仅是"比"请输出JSON"强10倍
- 一定要写"禁止添加解释"，否则AI喜欢加废话
- JSON是程序能直接解析的格式，是自动化的前提

**JSON基础：**
- 用`{}`包裹，里面是"键:值"对
- 键必须双引号，值可以是文字/数字/布尔
- 多个键值对用逗号隔开
- 示例：`{"名字":"竹子","年龄":26}`

---

### 技巧3：长度限制

**目的：** 控制输出篇幅，适合摘要/简报场景

**套路：** 字数上限 + 条数上限 + 禁止额外文字

**实战：AI资讯摘要**
```
【规则】
1. 最多5条摘要，每条严格不超过50个汉字
2. 只输出编号列表，不要标题、不要评论、不要多余文字
3. 超过5条只取最重要的5条，不足5条按实际数量
4. 格式：1. xxx\n2. xxx\n3. xxx
```

**适用场景：** 摘要生成、简报推送、信息提炼

**关键点：**
- "严格不超过"比"尽量简短"有效
- 必须写"禁止多余文字"，否则AI喜欢加总结句
- 条数限制和字数限制要同时写

## 四、综合实战：闲鱼客服提示词（从零自写）

```
【角色】你是一个闲鱼自动回复客服，AI头像定制服务，9.9元/个
【规则】
1. 客户会主动联系你，提出需求
2. 回复价格9.9元，3个以上七折，交付24小时内
3. 使用方法：出示图片+提出修改方式或描述需求
4. 禁止闲聊，不回答与商品无关的内容
5. 禁止私自改变价格
6. 只输出编号列表，每段不超过50字
7. 格式：1. xxx\n2. xxx\n3. xxx
【示例】
用户：你好，做出头像需要些什么，多少钱，可不可以便宜点，你顺便帮我看看明天的天气怎么样
回复：
1.您好，需要您出示需要修改的图片，提出修改方式或者描述一段需求
2.各种风格头像都只需9.9元
3.买≥3个头像可打七折
4.不好意思，我不回答与商品无关的问题
```

**复盘：**
- 第一版：角色模糊、规则空洞、示例没填 → AI无法执行
- 迭代过程：加具体信息 → 补拒绝示例 → 加格式约束 → 定稿
- 教训：**提示词不是一次写对的，是测出来的**

---

## 五、核心心法

1. **具体 > 模糊** — "9.9元"比"价格优惠"强100倍
2. **禁止 > 允许** — 先写不能做什么，再写能做什么
3. **示例 > 规则** — 1个示例比3条规则管用
4. **迭代 > 完美** — 先写再测再改，别想一次到位
5. **自动化思维** — 格式强约束+长度限制=可程序化处理=自动化前提
